R Programming

Instruction Set 1

Follow these steps to install R:

  • Visit the official R website.
  • Download the version suitable for your operating system.
  • Install R and set up RStudio.

Instruction Set 2

How to write and run a basic R script:

  • Open RStudio.
  • Create a new script by clicking 'File' > 'New File' > 'R Script'.
  • Write your R code and click 'Run'.

Instruction Set 3

Using R libraries:

  • Install a library by using install.packages("package_name").
  • Load the library using library(package_name).
  • Explore the functions within the package by checking the documentation.
